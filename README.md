# Flux0

Crypto-Trading Discord Community — flux0.dev

## Website

Die Landing Page liegt unter `web/`. Einfach `web/index.html` im Browser öffnen oder auf Cloudflare Pages / Vercel deployen.

## Discord Bot

1. Dependencies installieren: `pip install -r requirements.txt`
2. `.env.example` zu `.env` kopieren und Token eintragen
3. Bot starten: `python bot.py`

### Bot Commands
- `/setup` — Account-Panel posten
- `/account` — Account-Status anzeigen
- `/breakout add ada` — Token zur Watchlist hinzufügen
- `/breakout setup #channel` — Alert-Channel setzen
