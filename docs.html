LumaSpace

Live

LumaSpace ist eine KI-gestützte Produktivitäts- und Planungsplattform für Teams und Einzelpersonen mit Fokus auf Realtime Collaboration und intelligente Task-Organisation.

Was ist LumaSpace?

LumaSpace kombiniert Task-Management, Focus-Sessions und Team-Koordination in einer einzigen Oberfläche. Die KI analysiert Workload und schlägt optimale Zeitpläne vor.

⚡
Realtime Collaboration — Änderungen sind sofort für alle Teammitglieder sichtbar
🧠
AI Planning — Automatische Priorisierung und Scheduling basierend auf Deadlines
🎯
Focus Mode — Deep-Work-Sessions mit automatischem Status und Do-not-disturb

Quickstart

LumaSpace ist eine Web-App — keine Installation notwendig.

1
Account erstellen

Gehe zu lumaspace.de und registriere dich kostenlos.

2
Workspace anlegen

Erstelle deinen ersten Workspace — solo oder im Team. Importiere bestehende Tasks aus Notion, Trello oder Linear.

3
Loslegen

Füge Tasks hinzu, aktiviere den Focus Mode und lass die KI deinen Tag planen.

Features

Task Orchestration

Tasks können in Projekten gruppiert, mit Abhängigkeiten verknüpft und automatisch priorisiert werden. Deadlines und Aufwandsschätzungen fließen in die KI-Planung ein.

Focus Sessions

Der Focus Mode startet eine Pomodoro-ähnliche Session, schaltet Ablenkungen aus und trackt produktive Zeit pro Task.

API

Die LumaSpace API befindet sich in Vorbereitung. Trag dich in den Discord ein, um als erster Zugang zu erhalten.

FlowWave

In Entwicklung

FlowWave ist ein Discord-Bot-Builder auf Basis von pycord v2. Er abstrahiert die Komplexität von Discord-Bots und ermöglicht schnelle Entwicklung mit wenig Boilerplate.

FlowWave setzt Python 3.10+ und pycord v2.x voraus.

Installation

Voraussetzungen

  • Python 3.10 oder neuer
  • pip (Python Package Manager)
  • Ein Discord Developer Account + Bot Token

Via pip installieren

bash 
# FlowWave und pycord v2 installieren
pip install flowwave
pip install py-cord==2.6.1

Aus dem Quellcode

bash 
git clone https://github.com/flux0-network/flowwave
cd flowwave
pip install -e .

Quick Start

Ein minimaler FlowWave-Bot in wenigen Zeilen:

python bot.py 
import flowwave

# Bot initialisieren
bot = flowwave.Bot(
    token="DEIN_BOT_TOKEN",
    prefix="!",
    description="Mein erster FlowWave Bot"
)

# Slash Command registrieren
@bot.slash_command(name="ping", description="Antwortet mit Pong!")
async def ping(ctx):
    await ctx.respond(f"🏓 Pong! Latenz: {round(bot.latency * 1000)}ms")

# Bot starten
bot.run()
bash 
python bot.py
# ✓ FlowWave v0.1.0 gestartet
# ✓ Eingeloggt als MeinBot#1234
# ✓ 1 Slash Command registriert

Slash Commands

FlowWave macht es einfach, Discord Slash Commands mit pycord v2 zu bauen.

Einfacher Command

python 
@bot.slash_command(name="hallo", description="Begrüßt einen Nutzer")
async def hallo(ctx, name: str):
    await ctx.respond(f"Hallo, {name}! 👋")

Command Groups (Cogs)

python cogs/moderation.py 
import flowwave

class Moderation(flowwave.Cog):
    """Moderations-Commands"""

    @flowwave.slash_command(description="Einen Nutzer kicken")
    @flowwave.has_permissions(kick_members=True)
    async def kick(self, ctx, member: flowwave.Member, reason: str = "Kein Grund angegeben"):
        await member.kick(reason=reason)
        await ctx.respond(f"✅ {member.mention} wurde gekickt. Grund: {reason}")

def setup(bot):
    bot.add_cog(Moderation(bot))

Cog laden

python bot.py 
bot.load_extension("cogs.moderation")
bot.run()

Events

Mit @bot.event kannst du auf alle pycord v2-Events reagieren.

python 
@bot.event
async def on_ready():
    print(f"✓ Bot bereit: {bot.user}")

@bot.event
async def on_member_join(member):
    channel = member.guild.system_channel
    if channel:
        await channel.send(f"Willkommen, {member.mention}! 🎉")

@bot.event
async def on_message(message):
    if message.author == bot.user:
        return
    await bot.process_commands(message)

UI Components

FlowWave bietet fertige Wrapper für Discord UI-Komponenten (pycord v2 Views).

Buttons

python 
import flowwave

class ConfirmView(flowwave.View):
    @flowwave.button(label="Bestätigen", style=flowwave.ButtonStyle.green)
    async def confirm(self, button, interaction):
        await interaction.response.send_message("✅ Bestätigt!")
        self.stop()

    @flowwave.button(label="Abbrechen", style=flowwave.ButtonStyle.red)
    async def cancel(self, button, interaction):
        await interaction.response.send_message("❌ Abgebrochen.")
        self.stop()

@bot.slash_command(description="Aktion bestätigen")
async def confirm_action(ctx):
    await ctx.respond("Bist du sicher?", view=ConfirmView())

Modals

python 
class FeedbackModal(flowwave.Modal, title="Feedback"):
    feedback = flowwave.InputText(
        label="Dein Feedback",
        style=flowwave.InputTextStyle.long,
        placeholder="Was denkst du?"
    )

    async def callback(self, interaction):
        await interaction.response.send_message(
            f"Danke für dein Feedback: {self.feedback.value}"
        )

Vollständiges Beispiel

Ein kompletter FlowWave-Bot mit mehreren Cogs:

python bot.py 
import flowwave
from flowwave.config import Config

config = Config.from_env()  # lädt BOT_TOKEN aus .env

bot = flowwave.Bot(
    token=config.token,
    intents=flowwave.Intents.all(),
    activity=flowwave.Game("Flux0 Community")
)

# Alle Cogs aus dem /cogs Ordner laden
bot.load_extensions_from("cogs")

@bot.event
async def on_ready():
    print(f"✓ {bot.user} ist online")
    print(f"  Guilds: {len(bot.guilds)}")
    print(f"  Commands: {len(bot.pending_application_commands)}")

bot.run()

LyqDex

In Entwicklung

LyqDex ist ein dezentrales Exchange-Protokoll (DEX) mit Fokus auf effiziente Liquiditätspools, transparente Onchain-Transaktionen und minimale Gebühren.

LyqDex ist in aktiver Entwicklung. Die API kann sich noch ändern. Feedback und Beiträge im Discord sind willkommen.

Erste Schritte

SDK installieren

bash 
# npm
npm install @lyqdex/sdk

# yarn
yarn add @lyqdex/sdk

Verbindung aufbauen

typescript index.ts 
import { LyqDex } from "@lyqdex/sdk"

const dex = new LyqDex({
  rpc: "https://rpc.lyqdex.io",
  chainId: "lyq-mainnet-1",
})

await dex.connect(wallet)

Liquidity Pools

Pool abfragen

typescript 
const pool = await dex.pools.get("LYQ/USDC")

console.log(pool.tvl)         // Total Value Locked
console.log(pool.apy)         // Annual Percentage Yield
console.log(pool.fee)         // Pool-Gebühr in %

Liquidität hinzufügen

typescript 
const tx = await dex.pools.addLiquidity({
  pool: "LYQ/USDC",
  amountA: "100",   // LYQ
  amountB: "500",   // USDC
  slippage: "0.5"  // 0.5%
})

console.log("TX Hash:", tx.hash)

API Reference

MethodBeschreibungReturn
dex.pools.list() Alle aktiven Pools Pool[]
dex.pools.get(id) Pool by ID abrufen Pool
dex.swap(opts) Token tauschen TxResult
dex.pools.addLiquidity(opts) Liquidität einbringen TxResult
dex.pools.removeLiquidity(opts) Liquidität abziehen TxResult
